CommFort WebChat provides a web-based chat CommFort.

== Description ==

CommFort WebChat is the plugin that helps to realize web interface for [CommFort chat](http://http://www.commfort.com). Web interface allows to chat using your favourite browser and supports the most important functions of the chat (channel choose, fast message send and receive, authorize under you nick, etc.). Web interface gives access to view the chat and send messages anonymously.

-=Features=-

* Emoticons
* User information
* Sending messages from the web interface to the server with the subsequent publication in the active channel
* Chat rooms
* AJAX technology to get new messages from the database and records sent messages to the database

== Installation ==

Before installation of the plugin your must install [CommFort server](http://www.commfort.com/en/download.shtml) and [WebChat plugin](http://www.steelrat.info/?download=webchat_commfort_plugin_5). After you configure CommFort plugin you can install WordPress plugin.
You can use the built in installer and upgrader, or you can install the plugin
manually.

1. Upload `commfort-webchat` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[cf_webchat]` in your templates
